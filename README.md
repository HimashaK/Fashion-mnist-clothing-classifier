# Fashion-mnist-clothing-classifier
An end-to-end Fashion-MNIST classification pipeline in Python, featuring data preprocessing, model training, and evaluation. Models include a multinomial logistic regression baseline and a CNN with regularization (dropout/L1/L2) to improve generalization. Results include test accuracy, confusion matrix insights, and documented findings.

<img width="565" height="250" alt="image" src="https://github.com/user-attachments/assets/883c4303-143f-4504-ae81-4a3d9338c81a" />

The motivation for this project is to develop a machine learning model that can automatically classify clothing items from images. With text-tospeech integration, this project can be applied in
assistive technology to help visually impaired users
recognize and differentiate clothing items more
independently.
